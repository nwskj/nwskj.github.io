MusicFree 本地安装

电脑安装包手机安装包
如果网络安装失败（安装地址服务器被地方运营商屏蔽），请下载这个安装包后，解压到本地，然后选择「从本地安装」选择这些文件进行安装。

MusicFree 安卓版配置方法

  1. 点击左上角按钮
  2. 点击【插件设置】
  3. 点击右下角加号
  4. 点击【从网络安装插件】
  5. 输入插件 URL 即可（插件链接以 .json 或 .js 结尾）

MusicFree PC 版配置方法

  1. 点击主界面左侧【插件管理】
  2. 点击顶部【从网络安装】
  3. 输入插件 URL 即可（插件链接以 .json 或 .js 结尾）
